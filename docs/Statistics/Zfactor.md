# Z-factor

The Z-factor is a measure of statistical effect size and is calculated as

$Z-factor = 1 - \frac{3(\sigma_s + \sigma_c)}{\abs{\mu_s-\mu_c}}$

where s designates sample and c designates controls.

It is used to asses the quality of an assay, where
- Z = 1 Perfect Assay
- 1 < Z > 0.5 = great assay
- 0.5 < Z > 0 = marginal assay.
- Z > 0 indicates screening is practically impossible.

These apply to data that follows a normal distribution.
